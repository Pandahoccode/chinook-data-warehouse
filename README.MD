
# 🎵 Chinook Data Warehouse - ETL Project

> **BUT SD University Project** - Data Warehousing & Business Intelligence

## 🎯 Project Goal
Transform the Chinook music database into a **star schema data warehouse** using Oracle Data Integrator (ODI) to enable music sales analytics and business insights.

## 📊 What I Built
- **ETL Pipeline:** Automated data extraction, transformation, and loading
- **Star Schema:** Optimized for business intelligence queries
- **Data Warehouse:** Centralized repository for sales analysis
- **ODI Workflows:** Professional ETL development with Oracle Data Integrator

## 🏗️ Architecture
```
Chinook Database → ODI ETL → Star Schema → Business Insights
     ↓              ↓           ↓             ↓
 Operational     Transform   Dimensions    Sales Analytics
    Data           & Load    & Facts       & Reporting
```

## 🛠️ Technical Stack
- **ETL Tool:** Oracle Data Integrator (ODI)
- **Database:** Oracle Database
- **Model:** Star Schema (Dimensions + Facts)
- **Focus:** Data modeling & ETL development
